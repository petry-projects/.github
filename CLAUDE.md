# CLAUDE.md — Petry Projects `.github` Repository

This file provides instructions for Claude Code when working in this repository.

## Development Standards

See [AGENTS.md](./AGENTS.md) for the organization-wide development standards that apply to this repository.

## Repository Context

This is the `petry-projects/.github` organization repository. It contains:

- **`AGENTS.md`** — Organization-wide AI agent development standards
- **`standards/`** — Compliance standards and audit rules for all repos
- **`scripts/`** — Utility scripts used by workflows

## Key Guidelines

- This repository defines the standards that all other `petry-projects` repositories must follow.
- Changes to `standards/` files affect compliance audits across the entire organization — review carefully.
- Workflow files in `.github/workflows/` run organization-wide automation (compliance audits, etc.).
